# Sausage Automsg

An advanced, highly customizable auto-messaging utility designed to manage and broadcast up to four independent messages across various chat channels. Perfect for Trade, LFG, and guild recruitment on Warmane.

---

## üöÄ Features
* **Multi-Slot System:** Manage up to 4 independent message profiles (Tabs) at the same time. Each slot has its own text, interval timer, and targeted channels.
* **Dynamic Channel Detection:** The addon automatically detects which global channels you are currently connected to (Channels 1-6), alongside standard standard Say and Yell options.
* **Independent Controls & Master Switch:** Turn individual messages on or off instantly using their respective tickboxes. Control the entire broadcast engine with a single "Master Start/Stop" button.
* **Smart Item Linking:** Fully supports Shift-Clicking items, spells, and achievements directly into your custom message box.
* **Fail-Safe Configuration:** A dedicated "Save Text" button ensures your text, interval, and channel settings are only saved and applied to the engine when you are absolutely ready.

## üõ† Installation

1. Download the latest version of the addon.
2. Extract the folder into your World of Warcraft directory:
   `World of Warcraft/Interface/AddOns/`
3. Ensure the folder name is exactly **SausageAutomsg** (Remove any `-master` or version suffixes from the folder name).
4. Restart the game.

---

**UI Elements & Workflow:**
* **Minimap Icon:** Click the Sausage icon to toggle the UI. Right-click and drag to move it around your minimap.
* **Tabs (Msg 1 - 4):** Switch between your 4 message profiles.
* **"Save Text" Button:** **IMPORTANT!** You must click this button to save any text, interval, or channel changes made in the current tab.
* **"Enable" Tickbox:** Instantly turn the current message profile ON or OFF (does not require saving).
* **Master Start/Stop:** The main engine switch. When running, any enabled message slot will broadcast according to its saved interval.
